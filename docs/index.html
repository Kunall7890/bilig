XLSX Cache Doctor for Node and CI

Catch stale XLSX formulas.

SheetJS, ExcelJS, and xlsx-populate can write the file and still leave old cached formula values behind. Bilig checks the workbook locally before your API, queue worker, CI job, or agent trusts it.

No-clone check npm exec --package @bilig/xlsx-formula-recalc@latest -- xlsx-cache-doctor --demo --json See the formula cell, stale cached value, fresh recalculated value, and exact cell address.

Who needs this

Teams that read XLSX formula results in Node.

The risky moment is boring: a script edits input cells, saves the workbook, then production reads a cached formula value that Excel never refreshed. Bilig makes that state visible.

first doorway Find stale formula caches before code trusts the file.

Run it locally, in CI, or as a report-only GitHub Action. You get the workbook path, formula cell, cached value, fresh recalculated value, and a JSON report a maintainer can inspect.

Market gap

Not another spreadsheet UI. Not just a formula engine.

File libraries move workbook bytes. Formula engines calculate formulas. Hosted spreadsheets need networked app state. Bilig targets the missing runtime layer for backend teams that need workbook-shaped logic under test.

01 / model

Keep the business rule readable.

Represent inputs, formulas, and outputs as sheets instead of burying every rule in custom service code.

02 / run

Let the service change the inputs.

Write values through a narrow API, recalculate dependent formulas, and read the cells that determine the decision.

03 / save

Persist the workbook state.

Save WorkPaper JSON for route handlers, queue jobs, tests, agent tools, and audit trails.

04 / bridge

Import or export XLSX when files matter.

Use the XLSX subpath and recalculation packages when the workflow still needs Excel-compatible files.

Quickstart

Run the cache doctor first.

No account, no server, no spreadsheet UI. The demo intentionally includes a stale cached value so the JSON report proves the check is doing work.

No-clone command

terminal Node.js 22+ 
npm exec --package @bilig/xlsx-formula-recalc@latest -- \
  xlsx-cache-doctor --demo --json

Need this in pull requests? Use the report-only GitHub Action. Need a service-owned workbook instead of an XLSX file? Run npm create @bilig/workpaper@latest pricing-workpaper. Need a file-backed example? Open examples/xlsx-recalculation-node.
